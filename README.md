# dickwyn.xyz

## Prerequisites

- **Node.js**: LTS version
- [**pnpm**](https://pnpm.io/installation)

## Quick start

```sh
git clone https://github.com/dickwyn/v5.git
cd v5
pnpm install
pnpm dev
```

## Code quality

```sh
# lint
# (type-aware by default)
pnpm lint
pnpm lint:type-check

# format
pnpm fmt
pnpm fmt:check
```
