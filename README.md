# dickwyn.xyz

## Prerequisites

- **Node.js**: LTS version
- [**pnpm**](https://pnpm.io/installation)

## Quick start

```sh
git clone https://github.com/dickwyn/v5.git
cd v5
pnpm install
pnpm dev
```
